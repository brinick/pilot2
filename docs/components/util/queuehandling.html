pilot.util.queuehandling.get_queuedata_from_job(queues)[source]¶

Return the queuedata object from a job in the given queues object. This function is useful if queuedata is needed from a function that does not know about the job object. E.g. the pilot monitor does not know about the job object, but still knows about the queues from which a job object can be extracted and therefore the queuedata.
