pilot.workflow.generic.interrupt(args, signum, frame)[source]¶

Interrupt function on the receiving end of kill signals. This function is forwarded any incoming signals (SIGINT, SIGTERM, etc) and will set abort_job which instructs the threads to abort the job.

Parameters

  • args – pilot arguments.

  • signum – signal.

  • frame – stack/execution frame pointing to the frame that was interrupted by the signal.

Returns
